Usmaniya Technologies - Static Website
This is a ready-to-deploy static website. Files:
- index.html (home)
- assemble.html (details & pricing contact)
- assets/css/styles.css
- assets/js/main.js
- assets/img/logo.svg

To deploy: upload folder to any static hosting (Netlify, Vercel, GitHub Pages) or serve via simple HTTP server:
python3 -m http.server 8000
